This release expands the `fulhash` module with CRC32 and CRC32C support, unifies the hashing stack on `hash-wasm` for consistent performance, and introduces high-level convenience helpers for multi-algorithm hashing and verification.

### New Features

#### Expanded Algorithms

- **CRC32**: Standard IEEE 802.3 polynomial implementation.
- **CRC32C**: Castagnoli polynomial implementation (optimized for storage/network checksums).
- **Unified Stack**: All algorithms (XXH3, SHA, CRC) now run on `hash-wasm`, removing the need for separate `crc-32` and `fast-crc32c` dependencies.

#### Convenience API

- **`multiHash(input, algorithms)`**: Efficiently compute multiple checksums in a single pass over the data stream. Ideal for generating both integrity (SHA256) and deduplication (XXH3) hashes simultaneously.
- **`verify(input, checksum)`**: Validate data against a formatted checksum string (e.g., `crc32:cbf43926`). Automatically detects the algorithm from the prefix.

#### Performance

Benchmarked on Apple Silicon:

- **XXH3-128**: ~5 GB/s (Streaming & Block)
- **SHA-256**: ~2 GB/s
- **CRC32/C**: ~1.2 GB/s
